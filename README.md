# userscripts

hideo54 が日常生活を便利にするために都度書いた userscripts 置き場だよ。**何の責任も取らないよ。**

## 使い方

使いたい `.user.js` を選択して "Raw" を押せば、Tampermonkey が開くはずだよ。
